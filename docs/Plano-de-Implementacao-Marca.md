# 🔥 Plano de Implementação — Marca Fenice Lab

> Como sair do Manual da Marca (v1) para o **aplicativo da Fenice (Sistema Central/Portal)** nascido com a identidade certa. A marca é a fundação; o app é o destino. A vitrine de marketing será feita com **prints das telas do app**, não com cases de cliente (decisão 01/06).
> **Princípio inegociável:** *trabalharemos sempre com a verdade* — toda promessa com prova, nenhuma alegação inventada. A transparência é o produto.

> 🎯 **NORTE:** este plano alimenta o **Sistema Central** (PRD em `00-Central/Sistema Central Fenice - Ideias e Base do PRD.md`). A marca não é fim em si — ela existe para o app nascer coerente. Cases de cliente ficam para depois; agora a prova visual = telas do produto.
